R3 + UniTask 是目前 Unity 开发中最强组合之一（尤其是 2024–2026 年主流高性能写法），由同一位作者（neuecc / Cysharp）打造，互补性极强。  
为Unity当前最主流的零GC/高性能响应式 + 异步栈。

简介：
* UniTask：负责高性能 **异步编程** （取代 Coroutine / System.Threading.Tasks）
* R3：负责高性能 **响应式编程** （取代 UniRx / 事件系统 / Update 广播）

本文简单记录一下如何部署 Unity R3+UniTask 组合。

核心部署要点：
1. 先安装NuGet
2. 安装R3+UniTask核心
3. 通过Git安装Unity Editor扩展

基础信息：
* Unity：2022.3.62f3

步骤 - 安装 **NuGet For Unity**：
1. 安装 **NuGet For Unity**：使用Install package from git URL 输入 `https://github.com/GlitchEnzo/NuGetForUnity.git?path=/src/NuGetForUnity`
2. 完成安装后菜单新增 `NuGet` 选项，使用 `NuGet - Manage NuGet Packages` 搜索指定包体安装

步骤 - 安装 R3+UniTask - NuGet：
1. NuGet - `R3` 核心
2. NuGet - `UniTask`
    * 如果你需要 UniTask 的 Unity 扩展（如 PlayerLoop 集成），可选再加 git URL（但 NuGet 版已够用大部分场景）：
    * `https://github.com/Cysharp/UniTask.git?path=src/UniTask/Assets/Plugins/UniTask`
3. NuGet - `ObservableCollections`
    * 先安装 `ObservableCollections` （核心集合库）
    * 再搜索并安装：`ObservableCollections.R3` （R3 专用扩展，提供 .Observe()、同步视图等）
    * **注意安装顺序！**先装 R3，再装这个扩展，否则会报类型找不到（如 Subject<> 或 Observable 相关）

安装完后的标准刷新步骤（防坑）：
* 关闭 Unity 编辑器
* 项目根目录删除 Library 文件夹（强制重建缓存）
* 重新打开项目，等 Reimport 全部完成（Console 没红错误）
* 如果用 Rider/Visual Studio：Regenerate project files（Edit → Preferences → External Tools）

在Assets/Packages中可以查看已经安装的包体，通常包含以下包体：
* Microsoft.Bcl.AsyncInterfaces.6.0.0
* Microsoft.Bcl.TimeProvider.8.0.0
* ObservableCollections.3.3.4
* ObservableCollections.R3.3.3.4
* R3.1.3.0
* System.ComponentModel.Annotations.5.0.0
* System.Runtime.CompilerServices.Unsafe.6.0.0
* System.Threading.Channels.8.0.0
* UniTask.2.5.10

**验证安装是否成功：**

把这个类放进 Unity 的 Tests 文件夹，打开 Window → General → Test Runner，运行 EditMode 测试即可。

```csharp
using NUnit.Framework;
using R3;
using ObservableCollections;
using ObservableCollections.R3;
using System;
using System.Collections.Generic;
using UnityEngine.TestTools;  // 如果是 Unity 测试，可选

public class ObservableListR3Tests
{
    private ObservableList<int> _list;
    private List<CollectionAddEvent<int>> _addedEvents;
    private List<CollectionRemoveEvent<int>> _removedEvents;
    private List<CollectionReplaceEvent<int>> _replacedEvents;
    private IDisposable _subscription;

    [SetUp]
    public void Setup()
    {
        _list = new ObservableList<int>();
        _addedEvents = new List<CollectionAddEvent<int>>();
        _removedEvents = new List<CollectionRemoveEvent<int>>();
        _replacedEvents = new List<CollectionReplaceEvent<int>>();

        // 订阅所有主要事件（使用 R3 的 ObserveXXX）
        _subscription = Disposable.Combine(
            _list.ObserveAdd().Subscribe(e => _addedEvents.Add(e)),
            _list.ObserveRemove().Subscribe(e => _removedEvents.Add(e)),
            _list.ObserveReplace().Subscribe(e => _replacedEvents.Add(e))
            // 可以继续加 ObserveMove(), ObserveReset() 等
        );
    }

    [TearDown]
    public void TearDown()
    {
        _subscription?.Dispose();
        _list.Clear();
    }

    [Test]
    public void Add_Item_TriggersAddEvent()
    {
        // Act
        _list.Add(42);
        _list.Add(100);

        // Assert
        Assert.That(_addedEvents.Count, Is.EqualTo(2));

        Assert.That(_addedEvents[0].Value, Is.EqualTo(42));
        Assert.That(_addedEvents[0].Index, Is.EqualTo(0));

        Assert.That(_addedEvents[1].Value, Is.EqualTo(100));
        Assert.That(_addedEvents[1].Index, Is.EqualTo(1));

        Assert.That(_list.Count, Is.EqualTo(2));
        Assert.That(_list[0], Is.EqualTo(42));
    }

    [Test]
    public void Remove_Item_TriggersRemoveEvent()
    {
        _list.Add(10);
        _list.Add(20);
        _list.Add(30);

        // Act
        var removed = _list.Remove(20);

        // Assert
        Assert.That(removed, Is.True);
        Assert.That(_removedEvents.Count, Is.EqualTo(1));
        Assert.That(_removedEvents[0].Value, Is.EqualTo(20));
        Assert.That(_removedEvents[0].Index, Is.EqualTo(1));  // 移除前的索引

        Assert.That(_list.Count, Is.EqualTo(2));
        CollectionAssert.AreEqual(new[] { 10, 30 }, _list);
    }

    [Test]
    public void Replace_Item_TriggersReplaceEvent()
    {
        _list.Add(5);
        _list.Add(15);

        // Act
        _list[1] = 25;  // 替换索引1的值

        // Assert
        Assert.That(_replacedEvents.Count, Is.EqualTo(1));
        Assert.That(_replacedEvents[0].OldValue, Is.EqualTo(15));
        Assert.That(_replacedEvents[0].NewValue, Is.EqualTo(25));
        Assert.That(_replacedEvents[0].Index, Is.EqualTo(1));

        Assert.That(_list[1], Is.EqualTo(25));
    }

    [Test]
    public void AddRange_And_Clear_TriggersMultipleEvents()
    {
        // Act: 添加多个
        _list.AddRange(new[] { 1, 2, 3, 4 });

        // Assert 添加事件（AddRange 通常触发多次 Add）
        Assert.That(_addedEvents.Count, Is.EqualTo(4));  // 逐个触发 Add

        // Act: 清空
        var oldCount = _list.Count;
        _list.Clear();

        // Assert 清空通常触发 ObserveReset 或多个 Remove，这里检查 Remove 事件数
        Assert.That(_removedEvents.Count, Is.EqualTo(oldCount));  // 每个元素触发 Remove
        Assert.That(_list.Count, Is.EqualTo(0));
    }

    [Test]
    public void Dispose_Subscription_StopsEvents()
    {
        _subscription.Dispose();
        _subscription = null;

        // Act
        _list.Add(999);

        // Assert: 订阅已Dispose，不再收到事件
        Assert.That(_addedEvents.Count, Is.EqualTo(0));
    }
}
```
